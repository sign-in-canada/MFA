Sécurisez votre compte

Préparez votre appareil mobile

Vous pouvez utiliser une application sur votre appareil mobile (téléphone intelligent ou tablette) comme second facteur.

  • Quelles applications suggérons-nous?

    De nombreuses applications fournissent ce service. Nous suggérons d’utiliser Google Authenticator, Microsoft Authenticator ou Authy, mais vous pouvez utiliser n’importe quelle application similaire. Ces applications devraient être gratuites.

Vous n’avez pas d’application?

Installez une application d’authentification gratuite sur votre appareil mobile.

  • Comment installer une application d’authentification
    1. Ouvrez la boutique d’applications sur votre appareil et cherchez « authentificateur ».
    2. Choisissez l’application d’authentification à utiliser.
    3. Installez l’application choisie.

Vous avez déjà installé une application d’authentification?

Ouvrez l’application d’authentification sur votre appareil et ajoutez votre compte d’AIPRP en ligne :

  1. dans la majorité des applications, appuyez sur le signe « + » pour ajouter un nouveau compte;
  2. sélectionnez l’option de balayage d’un code à barres ou d’un code QR.

Passez à la page suivante lorsque vous êtes prêt à balayer un code QR.
