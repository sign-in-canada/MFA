Ce service utilise une authentification à deux facteurs pour sécuriser votre compte.

Vous pouvez configurer votre appareil mobile (téléphone intelligent ou tablette) pour qu’il serve de second facteur, ou utiliser une clé de sécurité FIDO (Fast Identity Online ou identité rapide en ligne) U2F (Universal Second Factor ou second facteur universel).

  • Qu’est-ce qu’une authentification à deux facteurs?

    L’authentification à deux facteurs permet de confirmer l’identité d’un utilisateur en combinant deux facteurs différents :

    • ce que l’utilisateur connaît (un mot de passe);
    • ce que l’utilisateur détient (un objet physique).

    Imaginez une porte qui ne peut être déverrouillée qu’à l’aide d’une clé et d’un code secret numérique. Une personne se faisant passer pour vous ne devrait jamais être en mesure d’obtenir ces deux facteurs.

    Pour les services d’AIPRP en ligne, vous pouvez configurer votre téléphone intelligent pour qu’il serve de second facteur, ou utiliser une clé de sécurité FIDO U2F (dispositif de sécurité matérielle spécial qui se branche dans le port USB de votre ordinateur).

    Vous pouvez acheter des clés de sécurité FIDO U2F en ligne, ce qui constitue la meilleure option si vous n’avez pas de téléphone intelligent.

    Vous ne pouvez utiliser aucune de ces options?

    Veuillez communiquer avec l’équipe de soutien des Services numériques d’AIPRP pour obtenir de l’aide.
