This service uses secure 2-factor authentication to secure your account.

You can set up your mobile device (smartphone or tablet) to serve as a second factor, or you can use a FIDO ("Fast Identity Online") Universal Second Factor (U2F) security key.

  • What is 2-factor authentication?

    2-factor authentication is a way to confirm users by using a combination of two different factors:

    • something they know (for example a password)
    • something they have (a physical object)

    Think of it like a locked door that can only be opened with both a key and a secret code on a keypad. Someone pretending to be you should never be able to get both those things.

    You can set up your mobile device (smartphone or tablet) to serve as a second factor, or you can use a FIDO U2F security key (a special hardware security device that fits in the USB port of your computer).
